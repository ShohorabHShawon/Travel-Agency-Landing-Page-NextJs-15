# Travel Agency Landing Page

This is a modern travel agency landing page built with [Next.js](https://nextjs.org), bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Features

- Responsive design for all devices
- Interactive booking components
- Destination showcase
- Customer testimonials
- SEO optimized
- Fast loading performance

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

## Technologies Used

- **Next.js** - React framework
- **Tailwind CSS** - For styling
- **React Icons** - For UI icons
- **Next/font** - Using [Geist](https://vercel.com/font), a custom font family from Vercel
